# Cinema Manager 

Консольний застосунок для управління кінозалами та кіносеансами. 

## Архітектура та структура проєкту

Рішення розділене на 4 окремі проєкти (бібліотеки класів та консольний застосунок) згідно з принципом Single Responsibility:

### 1. `CinemaManager.Models` (Шар даних)
Містить класи, які відповідають виключно за зберігання даних.
* **Сутність 1-го рівня:** `CinemaHall` (Кінозал). Не містить посилань на сеанси, щоб уникнути жорсткої прив'язки.
* **Сутність 2-го рівня:** `MovieSession` (Кіносеанс). Зберігає лише ідентифікатор залу (`CinemaHallId`), якому він належить.
* **Енумератори:** `HallType` (тип залу) та `Genre` (жанр фільму).

### 2. `CinemaManager.UI` (Шар відображення)
Містить UI-моделі (View Models), які відповідають за форматування та відображення даних користувачу. 
* `CinemaHallUI`: Обгортка для `CinemaHall`. Тут реалізовано зв'язок 1-до-багатьох — клас містить список сеансів (`List<MovieSessionUI>`) та логіку підрахунку загальної тривалості.
* `MovieSessionUI`: Обгортка для `MovieSession` зі зручним форматуванням дати, часу та тривалості.

### 3. `CinemaManager.DB` (Шар доступу до даних)
Відповідає за роботу зі сховищем.
* `Database`: Статичний клас, який виконує роль штучного сховища. Ініціалізується початковими даними при першому зверненні (3 зали та 12 сеансів).
* `CinemaRepo`: Репозиторій для взаємодії зі сховищем. Надає методи для отримання всіх залів та пошуку сеансів за ідентифікатором конкретного залу.

### 4. `CinemaManager.ConsoleUI` (Консольний застосунок)
Головний проєкт для взаємодії з користувачем. Реалізує консольне меню, через яке відбувається звернення до репозиторію, завантаження сутностей, їх конвертація в UI-моделі та виведення на екран.

## Основний функціонал
* Отримання та відображення списку всіх кінозалів.
* Вибір конкретного залу користувачем.
* Автоматичне підвантаження сеансів для обраного залу.
* Відображення детальної статистики залу та розкладу його сеансів.
* Можливість багаторазового перегляду різних залів без перезапуску програми.

## Як запустити проєкт
1. Склонуйте репозиторій на свій комп'ютер.
2. Відкрийте файл рішення `CinemaManager.sln` у Visual Studio.
3. Переконайтеся, що `CinemaManager.ConsoleUI` встановлено як стартовий проєкт (StartUp Project).
4. Натисніть `F5` або кнопку `Start` для запуску застосунку.
5. Для коректної роботи потрібен встановлений .NET 10.
